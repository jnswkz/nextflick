# **App Name**: CineView

## Core Features:

- Film Browsing: Browse films by genre, new releases, and curated categories.
- Robust Search: Quickly find films by title, actor, or keyword.
- Integrated Video Player: Stream films with playback controls, subtitles, and quality adjustments.
- Personalized Watchlist: Add films to a watchlist for viewing later.

## Style Guidelines:

- Background color: Dark gray (#222222) for a modern, immersive feel in dark mode.
- Primary color: Deep purple (#673AB7) for branding and key elements.
- Accent color: Teal (#008080) for highlights, interactive elements, and a touch of elegance.
- Body font: 'Inter', a sans-serif font for clear and modern readability.
- Headline font: 'Space Grotesk', a sans-serif font that pairs well with 'Inter' and provides a slightly computerized feel.
- Use minimalist, modern icons for navigation and film controls.
- Subtle transition animations for a smooth and engaging user experience.
